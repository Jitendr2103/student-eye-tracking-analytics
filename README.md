# Eye Tracking Student Analytics

A real-time student attention monitoring system using a webcam, built with Python, OpenCV, and MediaPipe. It tracks eye movements and head positions to generate detailed analytics regarding student focus, distractions, and confusion levels, which can be visualized in a dedicated Teacher Dashboard.

## Key Features

- **Teacher Analytics Dashboard** — A beautiful, local web application (HTML/JS/CSS) to visualize student data over time. No cloud setup required!
- **Time-Series Logging** — Logs attention metrics every second, allowing educators to map student performance across a timeline (e.g., finding the exact moment a student got confused during a lecture).
- **Robust Tracking** — Uses MediaPipe Face Mesh iris landmarks, making it tolerant to head movements and compatible with most glasses.
- **5-Point Calibration** — Maps your physical screen to the camera's gaze vectors.
- **Floating Control UI** — A small "always-on-top" window lets you start/stop tracking seamlessly while viewing notes or slides.

## Architecture & Codebase

The monolithic script has been cleanly modularized into the `src/` directory:
- `main.py`: The entry point to run the application.
- `src/tracker.py`: Core orchestrator and time-series logger.
- `src/gaze.py`: MediaPipe landmark processing and gaze math.
- `src/calibration.py`: OpenCV UI overlay for calibration and affine transformations.
- `src/analytics.py`: Sliding-window analytics and debounce logic.
- `src/ui.py`: Tkinter floating control panel and OpenCV HUD.
- `dashboard/`: Contains the Educator web dashboard (`index.html`, `style.css`, `script.js`).

## Requirements

- Python 3.9+
- A working webcam
- A desktop environment (Windows, macOS, Linux) for Tkinter & OpenCV GUIs.

## Installation

1. Create a virtual environment (recommended):
   ```bash
   python -m venv venv
   # Windows
   venv\Scripts\activate
   # macOS / Linux
   source venv/bin/activate
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

### 1. Running the Tracker
```bash
python main.py
```
1. **Calibration:** Look at the green dots as they appear on your screen (Center, and 4 corners).
2. **Start:** Once calibration is done, a floating control window appears. Open your notes/document and press **START**.
3. **Stop:** Press **STOP** when the session is over. The application will automatically generate two files:
   - `report_TIMESTAMP.csv` (Session summary)
   - `timeseries_TIMESTAMP.csv` (Second-by-second analytics)

### 2. Viewing the Teacher Dashboard
1. Open the `dashboard/index.html` file in any modern web browser.
2. Click **Upload Student Data**.
3. Select the `timeseries_TIMESTAMP.csv` file generated by your tracking session.
4. View the interactive graphs showing attention drops, cumulative distractions, and confusion over time.
